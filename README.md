# 🌟easy-gpt-vue
利用vite+vue写出来简易的GPT网页，你可以自己DIY这个GPT网页，还能把DIV的网页打包成软件

# 🚀快速开始
## 🏠本地部署
下载代码
```bash
git clone git@github.com:MCheping8108/easy-gpt-vue.git
```

安装依赖
```bash
npm install
```

在本地开发(仅适用于开发)
```bash
npm run dev
```

编译
```bash
npm run build
```

开启服务器
```bash
npm run preview
```

打包软件
```bash
npm run app:build
```
